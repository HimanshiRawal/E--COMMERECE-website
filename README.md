
E-COMMERCE WEBSITE
Project Title: E-commerce Website
Technology Used: HTML, CSS, JavaScript
DESCRIPTION
This E-commerce Website is a front-end web project designed to display products, prices, and basic shopping features. The website provides a user-friendly interface where users can browse products, view details, and contact the store. It is suitable for learning web development basics and understanding how online shopping platforms work.
FEATURES


Home page with product highlights


Product listing section


Price display for items


Search bar for products


Responsive navigation menu


Contact form for user inquiries


Clean and modern UI design


FOLDER STRUCTURE


index.html → Main homepage


css/ → Contains all CSS files


js/ → Contains JavaScript files


images/ → Contains images used in the website


README.txt → Project documentation


HOW TO RUN


Download or clone the project files


Open the folder


Double-click on index.html


The website will open in your browser
